HTML 教学教材

面向初学者的结构化学习指南

1. HTML 是什么？

HTML（HyperText Markup Language）是网页的骨架，用来描述内容结构。

  • 结构：标题、段落、列表、表格。
  • 语义：让搜索引擎和浏览器理解内容。

2. 基本文档结构

每个 HTML 页面都包含以下关键标签：

<!DOCTYPE html>
<html lang="zh-cn">
  <head>
    <meta charset="UTF-8" />
    <title>页面标题</title>
  </head>
  <body>
    <h1>我的第一张网页</h1>
  </body>
</html>

3. 常用标签

以下是最常见的内容标签：

  • <h1>~<h6>：标题等级。
  • <p>：段落。
  • <a>：超链接。
  • <img>：图片。

4. 列表与表格

列表与表格帮助我们组织信息。

<ul>
  <li>无序列表项目</li>
  <li>另一个项目</li>
</ul>

<table>
  <tr>
    <th>姓名</th>
    <th>职业</th>
  </tr>
  <tr>
    <td>李雷</td>
    <td>开发者</td>
  </tr>
</table>

5. 表单基础

表单用于收集用户输入。

<form>
  <label>姓名：<input type="text" name="name" /></label>
  <button type="submit">提交</button>
</form>

6. 学习建议

  1. 先学结构，再学样式（CSS）。
  2. 多练习：模仿真实网页布局。
  3. 善用浏览器开发者工具。
